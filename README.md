# Property Admin Dashboard

A comprehensive, unique white-themed admin dashboard for managing properties, blogs, team members, and contact leads.

## Technology Stack
- **React 18** (Vite)
- **Tailwind CSS** (Clean White Theme)
- **React Router v6**
- **Axios** (API Management)
- **React Toastify** (Notifications)
- **React Icons**

## Features
- **Dashboard**: Overview and statistics.
- **Properties**: Full CRUD with image upload, featured toggle, and advanced search.
- **Blogs**: Manage blog posts with categorical filtering and publish toggles.
- **Categories**: Simple blog category management.
- **Team**: Manage team members with profile images and active status.
- **Contacts**: Track leads, view messages, and update follow-up status.

## Getting Started

1.  Navigate to this directory:
    ```bash
    cd property-admin-dashboard
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Configure your backend URL in `.env`:
    ```
    VITE_API_BASE_URL=http://localhost:5000/api
    ```

4.  Run the development server:
    ```bash
    npm run dev
    ```

## Design Principles
- **Clean White Theme**: No gradients, minimal use of shadows, and high-quality typography.
- **Premium UI**: Smooth transitions, consistent spacing, and clear interactive states.
- **Robust Error Handling**: Real-time feedback via toasts and clear validation messages.
